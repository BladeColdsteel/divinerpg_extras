 Welcome to DivineRPG Extras! 

This mod is an addon to DivineRPG and that adds more functionality to DivineRPG!

It adds Tinker's Construct Functionality currently with all DivineRPG materials having toolparts created.

I will add more weapons/tools to the mod and the plan is to add structures and ores as well to the Twilight Dimensions, but this was my first mod and I wanted to post it to keep track of my progress.
